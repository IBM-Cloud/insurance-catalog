Node.js Catalog RESTful API

GET /items to see all the items.

GET /items/{id} retrieves an item from the DB using an ID.

GET /db/{option} Create (and populate) or delete the DB. Option is either 'create' or 'delete.'

GET /fib Computes the Fibonacci sequence of 20.

PUT /items/{id} updates an item using an ID.

DELETE /items/{id} removes an item from the DB using an ID.

POST /items creates an new item.
  Ex: {name: 'War Room Table',color: 'tan', quantity: 5, description: 'A Beautiful table.',usaDollarPrice: 180.00, imgsrc:'http://image.jpg'}
